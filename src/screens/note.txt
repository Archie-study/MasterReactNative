Meeting 5 - Add Custom Font and Icon Into A Popular Layout
Font Assets : https://bit.ly/app1_meeting5_font
Image Assets : https://bit.ly/app1_meeting5_img

NOTE : 
"npx react-native link" is not valid anymore, so try this one :
"npx react-native-asset"

To Install React Native Vector Icons
1. npm install react-native-elements
2. npm install --save react-native-vector-icons
3. npx react-native-asset

See The Icons Library
1. https://oblador.github.io/react-native-vector-icons/
2. https://reactnativeelements.com/docs/components/icon


Don't forget to create react-native.config.js file and type this code :
//react-native.config.js
module.exports = {
    assets: [
        './assets/fonts',
        './node_modules/react-native-vector-icons/Fonts'
    ]
}

==> After that, run 'npx react-native-asset' again
